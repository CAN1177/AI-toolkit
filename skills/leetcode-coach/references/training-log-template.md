# 训练记录模板

在一轮训练收尾时，或者学习者开启新对话、想避免重复做题时，使用这个模板。

## 单次训练总结

```text
题目：
模式：
结果：做出来了 / 做对一部分 / 卡住了
关键失误：
正确触发信号：
还需要补练的点：
下一题建议：
```

## 新对话历史摘要

让学习者贴一段像下面这样的简短历史：

```text
最近训练：
1. 题目 / 模式 / 结果
2. 题目 / 模式 / 结果
3. 题目 / 模式 / 结果

目前还觉得薄弱的点：
更偏好的提示方式：
目标语言：
近期面试安排：
```

## 教练收尾格式

收尾时按这个顺序简洁输出：

1. `模式：` 这道题本质上在考什么
2. `触发信号：` 下次什么时候该想到这个模式
3. `主要失误：` 学习者今天最大的遗漏是什么
4. `下一步：` 一道邻近题，或者一个相邻模式
